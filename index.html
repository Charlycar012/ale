Calibración: 0.32 µm/pixel | Aumentos: 4X-10X

EPIDERMIS

Evaluación de biocompatibilidad superficial

1

Grosor Epidérmico

• Aumento: 10X
• Herramienta: Líneas perpendiculares (mínimo 3)
• Desde: Capa basal → Estrato córneo
• Nombre ROI: "Epidermis_ROI"
• Nombre líneas: "Base_1", "Base_2", etc.
Interpretación:
✓ Grosor normal = Biocompatible
✗ Muy grueso = Hiperplasia/Irritación
✗ Muy delgado = Atrofia/Daño
2

Continuidad Epidérmica

• Aumento: 4X-10X
• Herramienta: Polígono
• Clases: "Epidermis_presente" / "Epidermis_ausente"
• Cálculo: % pérdida = (Área ausente / Área total) × 100
Clasificación de severidad:
0 - Intacta: 0-1%
1 - Leve: 1-10%
2 - Moderada: 10-30%
3 - Severa: >30%
EVALUACIÓN DE TOLERANCIA EPIDÉRMICA

DERMIS

Evaluación de respuesta tisular profunda

1

Anotaciones Manuales

• Herramienta: Polígono
• Múltiples áreas de:
- Infiltrado inflamatorio (núcleos densos, morados)
- Colágeno (fibras rosadas, eosinofílicas)
- Edema (espacios claros entre fibras)
2

Entrenamiento del Clasificador

• Menú: Pixel Classification
• Acción: Train Pixel Classifier
• Aprendizaje: Patrones cromáticos y texturales
• Validación: Live Prediction
3

Cuantificación Automática

• Aplicar clasificador a dermis completa
• Exportar áreas relativas (%)
Interpretación:
• Infiltrado ↑ = Irritación
• Colágeno organizado = Estabilidad
• Edema ↑ = Inflamación

DERMIS

Análisis de remodelación tisular

1

Delimitación "Dermis_ROI"

• Herramienta: Polígono
• Incluir: Matriz extracelular + células
• Preparación: Estimate stain vectors (H&E)
2
Parcial

Detección de Células

• Menú: Cell detection
Parámetros:
- Detection: Hematoxylin OD
- Background radius: 8-10 µm
- Nuclear area: 20-200 µm²
- Threshold: 0.2-0.3
3
Parcial

Clasificación y Cuantificación

• Colágeno: ✓ Organización/densidad
• Fibroblastos: ✗ Área 30-70 µm², elongación ≥1.5
• Vasos: ✗ Angiogénesis
• Macrófagos: ✗ 100-250 µm², núcleo arriñonado
• Linfocitos: ✗ 30-60 µm², circularity >0.85
